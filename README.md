# Regalo para Kiara

Página romántica responsive creada para una historia entre Perú y Argentina.

## Enlace público
Después de publicar, el sitio estará en:
https://EDWIN913445.github.io/regalo-para-kiara/

## Actualizar fotos
1. Abre esta carpeta con Visual Studio Code.
2. Reemplaza archivos dentro de assets o edita index.html.
3. La canción se carga desde el reproductor oficial de YouTube.
4. Ejecuta en PowerShell dentro de la carpeta:

   git add .
   git commit -m "Actualizar regalo"
   git push

Nota: publica únicamente fotografías que tengas permiso de compartir.
